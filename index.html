Keyboard Shortcuts

Built for IT pros, writers, and power users. Skip the mouse — every action has a key.

Global (anywhere in Chrome)

Open the popup Alt + Shift + N
Clip selected text as note Alt + Shift + S
💡 Customize at chrome://extensions/shortcuts

Inside the popup

New note Ctrl + N
Save current note Ctrl + S
Bold / Italic / Underline Ctrl + B / I / U
Undo / Redo Ctrl + Z / Y
Exit prompter mode Esc

On macOS, use ⌘ instead of Ctrl.

How It Works

1

Install (30 seconds)

Add the extension from Chrome Web Store. No account needed, no permissions beyond what's necessary for the web clipper to work.

2

Set Master Password (optional, 1 minute)

Create a master password for AES-256 encryption — or skip it and use the extension unencrypted. Your call.

3

Clip the web as you browse

See something useful? Highlight it, right-click, save. Or press Alt+Shift+S. The page URL is attached automatically.

4

Search, organize, back up

Real-time search across all notes. Pin the important ones. Export an encrypted backup whenever you want.
